## üéØ Quick Facts

### Database
- **Type:** PostgreSQL (Supabase)
- **Tables:** 5 (auth.users, user_roles, profiles, cars, orders)
- **Security:** Row-Level Security enabled

### Authentication
- **Type:** JWT (Supabase Auth)
- **Roles:** admin, user
- **Default Admin:** admin@prosperousmotors.com

### Frontend
- **Framework:** React 18.3.1
- **Language:** TypeScript
- **Styling:** Tailwind CSS + shadcn/ui
- **Build Tool:** Vite

### Real-Time
- **Technology:** Postgres change subscriptions
- **Updates:** Instant to all clients
- **Coverage:** Cars, Orders, User Roles
