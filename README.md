# Prosperous Motors

Premium car dealership in Abuja offering luxury vehicles with trust, transparency and excellence.

## Project Overview

Prosperous Motors is a luxury car dealership platform built with modern web technologies to provide an exceptional shopping experience for premium vehicle buyers.

**Website**: https://prosperousmotors.com

## How can I edit this code?

### Use your preferred IDE

Clone this repo and work locally using Node.js & npm - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

Follow these steps:

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd prospermotors

# Step 3: Install dependencies
npm i

# Step 4: Start the development server
npm run dev
```

### Edit a file directly in GitHub

- Navigate to the desired file(s).
- Click the "Edit" button (pencil icon) at the top right.
- Make your changes and commit.

### Use GitHub Codespaces

- Navigate to the main page of your repository.
- Click on the "Code" button (green button) near the top right.
- Select the "Codespaces" tab.
- Click on "New codespace" to launch a new Codespace environment.
- Edit files and commit your changes.

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase (Backend)

## How can I deploy this project?

Deploy to Vercel, Netlify, or any Node.js hosting platform:

```sh
npm run build
npm run preview
```

Follow your deployment provider's documentation for pushing to production.

## Project Structure

- `/src` - Application source code
- `/src/pages` - Page components
- `/src/components` - Reusable components
- `/src/contexts` - React contexts for state management
- `/src/hooks` - Custom React hooks
- `/src/integrations` - External service integrations
- `/public` - Static assets
